# 📊 Furniture Sales Dashboard (Excel)

## 📌 Overview

This project analyzes real-world furniture sales data and transforms it into an interactive dashboard to support business decision-making.

## 🎯 Objectives

* Track sales performance
* Analyze customer segments
* Evaluate delivery efficiency
* Identify top-performing products

## 📊 Key KPIs

* Total Revenue
* Avg. Order Value (AOV)
* Orders by Region
* On-Time Delivery Rate
* Payment Methods Distribution

## 📈 Key Insights

* Individuals contribute the majority of sales (72%)
* Cash is the dominant payment method (51%)
* 32% of deliveries are delayed, indicating operational inefficiencies
* Sales are concentrated in a few top products

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning & Transformation
